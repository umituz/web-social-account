# Skills Directory

Bu dizin, @umituz/web-social-account paketinin nasıl kullanılacağını açıklar.

## 📁 Dosyalar

- **SKILL.md** - Ana skill dokümantasyonu
- **INTEGRATION.md** - Entegrasyon rehberi

## 🎯 Hızlı Başlangıç

```bash
# Paketi yükle
npm install @umituz/web-social-account

# Environment variables ayarla
cp .env.example .env

# Dokümantasyonu oku
cat skills/SKILL.md
```
